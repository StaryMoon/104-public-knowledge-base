- 对于`c = a`之后`d = a`，没有考虑两个`a`是同一个指针，直接新建一个对象导致`c`和`d`分离
- 对于`d = {1: d[1], 2:2]}`这种结构，没有考虑前后是不同指针，直接去`d[2] = 2`
- `elif`机制
```python
a = 1
if a == 1:
    a = 2
elif a == 2:
    a = 3
```
结果是`a = 2`
- 对于
```python

    if bar:
        ...
```

```python
if foo:
    if bar:
        ...
```
